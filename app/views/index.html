Requirements and information

This prototype form should take around 5 minutes to complete.

You should not input any real patient identifiable data into this prototype.

This prototype allows you to model the journey for:

  • an encounter with a form already completed (01/01/2024)
  • an encounter which has not had a form already completed (31/12/2023)
